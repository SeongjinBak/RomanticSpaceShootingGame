동하야! 별 따줄게
===============
**동하야! 별 따줄게** 는 **2018년 9월 27일** 부터 **2019년 11월 22일**까지, 홍익대학교 게임동아리 **ExP Make** 18-2학기 협동스터디에서, 기획자 2, 프로그래머 3, 아트 2명이 참여한 프로젝트 입니다.

이 프로젝트에서 게임플레이 프로그래밍, 애니메이션 프로그래밍을 담당하였습니다.
